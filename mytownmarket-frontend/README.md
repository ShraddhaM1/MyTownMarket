# MyTownMarket

MyTownMarket is a full-stack marketplace platform where local businesses can list their products and customers can browse, order, and pay using QR-based UPI payments.

## Features

### Customer Side
- Browse local businesses
- View products
- Add to cart
- Checkout with COD or UPI payment

### Admin Side
- Create businesses
- Upload business images
- Upload QR code for payments
- Add/Edit/Delete products
- View orders

## Tech Stack

Frontend:
- Next.js
- React
- Tailwind CSS

Backend:
- Node.js
- Express

Database:
- MongoDB

Image Upload:
- Multer

Authentication:
- JWT

## How to Run

### Backend
cd Backend
npm install
npm run dev

### Frontend
cd mytownmarket-frontend
npm install
npm run dev
