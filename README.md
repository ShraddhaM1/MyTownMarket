# MyTownMarket

MyTownMarket is a full-stack local business marketplace platform where users can browse businesses, view products, place orders, and pay using UPI QR or Cash on Delivery.

## Tech Stack
Frontend: Next.js, React  
Backend: Node.js, Express  
Database: MongoDB  
Authentication: JWT  

## Features
- User signup/login
- Business listing
- Product management
- Admin dashboard
- Shopping cart
- UPI QR payment
- Order management

## Project Structure
Backend → Express API  
Frontend → Next.js application

## Run Backend
cd Backend
npm install
npm run dev

## Run Frontend
cd mytownmarket-frontend
npm install
npm run dev
